## Installation

1. Make sure you've got:
    * Ruby
    * Python
    * Node.js
    * Bower
    * Gulp
2. Run following commands in main directory:
```
bower install
npm install
```
3. Run gulp and then gulp server to preview:
```
gulp
gulp serve
```
